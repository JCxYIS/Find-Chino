# 文言。尋找智乃
子安知智乃之何處乎？

## 用法
- 於第二行，可書欲尋智乃之句。後稱其為「欲尋之句」。
- 卿可依吾人所批之註，解文之意

## 終而得
- 壹、夫若智乃非存在以「欲尋之句」者，乃得「逝者已矣，來者可追。」勿縱心而悽。安其身，而莫怪云。

    > 噫！微斯人，吾誰與歸？

- 貳、夫若智乃存在以「欲尋之句」者，乃得其所恃之字元。其餘難以枚述，故粗載之。
